# anthropic

anthropic API bindings for Hot.

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/anthropic": "1.0.4"
```

## API Base URL

`https://api.anthropic.com`

## Documentation

Full documentation available at [hot.dev/pkg/anthropic](https://hot.dev/pkg/anthropic)
