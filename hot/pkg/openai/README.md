# openai

openai API bindings for Hot.

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/openai": "1.0.4"
```

## API Base URL

`https://api.openai.com/v1`

## Documentation

Full documentation available at [hot.dev/pkg/openai](https://hot.dev/pkg/openai)
