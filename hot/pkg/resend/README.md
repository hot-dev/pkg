# resend

resend API bindings for Hot.

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/resend": "1.0.3"
```

## API Base URL

`https://api.resend.com`

## Documentation

Full documentation available at [hot.dev/pkg/resend](https://hot.dev/pkg/resend)
